# AI App Builder Guide

An interactive visual guide to building web and mobile apps using AI assistants — Claude and OpenAI Codex.

## What's Inside

- **Claude vs Codex comparison** — interactive tabs explaining each AI tool
- **Step-by-step workflow** — Web App and Mobile App visual walkthroughs with copy-ready prompts
- **Tools ecosystem** — expandable cards for Claude, Cursor, Copilot, Bolt.new, v0.dev, Lovable, Windsurf, Replit
- **Architecture diagram** — visual flow from idea to deployed app
- **Interactive prompt builder** — fill in your app details and get a ready-to-use AI prompt

## Tech

Pure HTML/CSS/JS — no build step, no dependencies. Just open index.html.
